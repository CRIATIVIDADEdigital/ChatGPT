# Iniciação
Parabéns para você por ser curioso ou curiosa em aprender sobre o uso do chatgpt. Para que você possa alcançar o próximo nível de utilização do ChatGPT desafio a você a passar por esta iniciação.

É comum quando estamos falando sobre uma nova ferramenta ou conceito iniciar com algumas definições. É como o vocabulário base que utilizamos na área. Para o caso do ChatGPT considero relevante que você saiba o significado de alguns termos:
* ChatGPT;
* Prompt;
* LLM.

Mas eu não vou fazê-lo passar por um processo de leitura com a definição ou explicação de cada um dos termos dessa lista. Vou aproveitar que o ChatGPT é uma ferramenta capaz de gerar respostas textuais a partir das nossas perguntas, e usar o próprio CHatGPT para nos apresentar as definições.

:::tip

A seguir vou sugerir diversas perguntas que podem ser feitas ao ChatGPT. Minha sugestão é que você escolha as que achar mais interessante e experimente você mesmo o resultado. Use o ícone de *copiar* para que você possa colar o texto da pergunta no ChatGPT ou use o ícone *resposta ChatGPT* para ver o que o ChatGPT respondeu para mim quando fiz esta pergunta para ele.

:::

## ChatGPT
Então, vamos começar pedindo ao ChatGPT que fale sobre ele mesmo.
```url link='https://chatgpt.com/share/3de5759c-58fe-4395-bf6c-35e8ffc45ad9'
O que é ChatGPT?
```
Experimentou a pergunta acima? O que achou da resposta? O interessante de fazer este tipo de questionamento para o ChatGPT é que, se a resposta não nos atende, podemos ajustar a nossa pergunta em busca de uma resposta mais apropriada. Por exemplo, se achamos que a resposta ficou muito técnica, podemos incluir este fator em nosso texto, como abaixo.

```url link='https://chatgpt.com/share/cf22a64e-8c44-4fde-8e40-e6f0a99af36b'
Sou leigo em inteligência artificial e gostaria de entender como o ChatGPT funciona.
```
Não sei como o ChatGPT respondeu para você o pedido acima, mase se você pressionar o botão <IconExternalLink /> presente na pergunta acima (toque ou passe o mouse sobre ela para habilitar o botão). Ainda achei que ele usou termos técnicos além da resposta ficar longa.

Mas podemos controlar isso adicionando mais informações ao nosso texto de questionamento, veja esta nova opção abaixo.
```url wordWrap link='https://chatgpt.com/share/a86dc5d9-c14d-4c96-a38e-2190551b8ee3'
Sou leigo em inteligência artificial e tenho interesse em aprender como o ChatGPT funciona. Pode explicar de uma maneira que eu consiga entender, mesmo sem ter conhecimentos técnicos? Gostaria de uma explicação simples e concisa.
```
:::warning

Uma característica comum do ChatGPT é a de apresentar respostas diferentes para uma mesma pergunta, mais a frente no guia explicarei como isso funciona, portanto mesmo que você copie exatamente a pergunta aqui do guia a resposta do chatGPT pode ser diferente.

:::

## Prompt
Vamos seguir com o conceito de *prompt*. Novamente vamos recorrer ao ChatGPT. Vou sugerir alguns textos que você pode utilizar para explorar o conceito de *prompt* usando o próprio ChatGPT.

```url wordWrap link='https://chatgpt.com/share/5d8411f1-455a-4dd2-82b7-04f1c3064be6'
O que é um prompt?
```
```url wordWrap link='https://chatgpt.com/share/a8d61274-27f7-4826-b1de-110d111e3efe'
Explique detalhadamente o que é um prompt no ChatGPT, usando linguagem simples e exemplos práticos para facilitar a compreensão de iniciantes.
```
```url wordWrap link='https://chatgpt.com/share/60bb661c-d4ad-4e4e-b632-fe8154c1b676'
Você é um professor universitário de tecnologia explicando para uma turma de alunos iniciantes o que é um 'prompt' no ChatGPT. Sua tarefa é explicar de forma simples e didática, contextualizando como o ChatGPT usa o prompt para gerar respostas baseadas nas instruções fornecidas. Estruture a explicação em três parágrafos curtos e inclua um exemplo prático no final.
```
Agora que você já sabe o que é *prompt* passarei a usar esta palavra nos textos do guia.
